## Быстрый старт

### 1. Настройте контакты

Замените в `index.html`:
- `https://t.me/lovebatory` → ваш Telegram
- `hello@webhapa.ru` → ваш email
- `+7-900-123-45-67` → ваш телефон

### 2. Загрузите на хостинг

- Включите HTTPS
- Настройте кэширование
- Проверьте скорость загрузки

### 3. Настройте аналитику

- Яндекс.Метрика
- Google Analytics
- Настройте цели
