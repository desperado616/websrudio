## ⚙️ Настройка

### 1. Контакты

Замените в `index.html`:

```html
<!-- Telegram -->
https://t.me/yourstudio

<!-- Email -->
hello@webstudio.ru

<!-- WhatsApp -->
https://wa.me/79001234567
```

### 2. Название студии

Замените "WebStudio" на название вашей студии в:
- Логотип в header
- Логотип в footer
- Title страницы

### 3. Цвета

В `styles.css` измените переменные:

```css
:root {
    --primary: #6366f1;  /* Основной цвет */
    --primary-dark: #4f46e5;  /* Темный вариант */
}
```

### 4. Favicon

Замените emoji в `<link rel="icon">` на свой:

```html
<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>💻</text></svg>">
```
